利用者ガイド

プロンプトでフォームを作成し、投稿データを帳票化してAI分析を行う手順です。

1. フォームを作成する

  1. 「フォーム作成」へ移動します。
  2. プロンプトに作りたいフォームの内容を入力します。
  3. 「フォームを生成」でフォームを作成し、内容を確認します。
  4. 「業務フォームとして保存」でフォームを保存します。

例: 「経費精算フォームを作成して。項目は経費発生日、経費金額、経費カテゴリ」

2. データを投稿する

生成したフォームに沿ってデータを入力し、投稿します。投稿データは帳票分析で集計されます。

3. 帳票を分析する

  1. 「帳票分析」へ移動します。
  2. 利用可能なデータ（テーブル）を確認します。
  3. プロンプトで集計内容を指示します。
  4. グラフと一覧、AI分析結果を確認します。

4. プロンプトで分析を指示する

使用するテーブルや集計方法、グラフ形式はプロンプトで指定します。

例: 「sales_table を使って月別売上推移を折れ線グラフで表示」

5. AI分析結果を確認する

「傾向を教えて」などの指示で、現在表示中のデータに対する分析を実行できます。

よくある質問

  • グラフが出ない: フォームの投稿データがあるか確認してください。
  • 日付が1日しか出ない: 横軸が「送信日」になっていないか確認してください。
  • AI分析が出ない: 「傾向」「原因」などの指示を含めてください。
